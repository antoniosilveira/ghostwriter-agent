**Purpose:** Draft cold or warm outreach to a specific person.

**Input:** Name of the person + brief context. Example: "Sarah Chen, CTO at Beacon Labs — want to connect about AI in engineering." More context = better draft.

**Workflow:**
1. Determine channel from context:
   - LinkedIn DM or InMail for professional/cold contacts
   - iMessage or text for personal contacts or former colleagues
   - Email if specified
2. Draft using VOICE.md outreach patterns:
   - Open with personal connection or specific reference — not generic flattery
   - State the ask as a favor, not a demand
   - Frame it around learning, not extracting time
   - Give them an out: "No pressure if it's not the right timing"
   - Sign-off: `thanks in advance, /[Name]`
3. Output one version unless channel is unclear, in which case offer two

**Output format:**
```
Channel: [LinkedIn DM / iMessage / Email]

[draft]

---
Context used: [brief note on what informed the draft]
```

**Constraints:**
- No em dashes
- No generic flattery ("I'm a huge fan of everything you do")
- Specific reference always beats a generic compliment — if no specific reference exists, ask for one before drafting
- Do not fabricate prior relationship context
